Список использованных команд в терминале:

Переход в галлерею
  • git —version - проверка версии
  • git init - инициализация
  • git status - проверка статуса
  • git add - добавление файлов на отправку
  • git commit - m "content" - создание комментария
  • git branch - проверка списка веток
  • git checkout -b "content" - создание ветки
  • git merge - объединение контента из ветки
  • git branch -d - удаление ветки
  • git log - история изменений
  • git push - запушить файлы в репозиторий
  • git pull - стянуть файлы с репозитория
  • git clone - скачать копию репозитория
  • ls - посмотреть список файлов
  • pwd - посмотреть где ты находишься
  • cat - открыть файл
  • touch - создать файл
  • rm - удалить файл
  • clear - очистить терминал
  • cd - переход между папками
  • mkdir - создать папку
